# Music File Instructions

## Required Audio File

Please add your audio file here:
- **File name:** `blue-instrumental.mp3`
- **Full path:** `/app/frontend/public/music/blue-instrumental.mp3`
- **Song:** Blue - Yung Kai (Instrumental)

## How to Add the Audio File

1. Download or obtain the "Blue - Yung Kai Instrumental" MP3 file
2. Place it in this directory: `/app/frontend/public/music/`
3. Name it exactly: `blue-instrumental.mp3`
4. Restart the frontend: `sudo supervisorctl restart frontend` (optional, but recommended)

## File Specifications

- Format: MP3
- Quality: Any (will be played at 60% volume)
- The music will:
  - Loop continuously
  - Start automatically after unlock animation
  - Have a play/pause toggle button in bottom-right corner
